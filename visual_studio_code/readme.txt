# https://github.com/deoren/config-files

The settings.json file should be placed in these paths for the stable version
of VS Code:

- Windows: %APPDATA%\Code\User
- Linux: $HOME/.config/Code/User

or in these locations for the Nightly/Unstable/Insiders version:

- Windows: %APPDATA%\Code - Insiders\User
- Linux: $HOME/.config/Code - Insiders/User
